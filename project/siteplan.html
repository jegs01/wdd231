Site Plan for Bird Species

Site Name

The site name is birdspecies.comThis name was selected because it effectively conveys the website's purpose, which is to display the incredible diversity of bird species around the globe.

Site Purpose

The site aims to offer a rich, engaging experience for learning and discovery.

Scenarios

Scenario 1: An individual who has been bird-watching for several years is keenly interested in learning more about the different species they encounter.

Scenario 2: An individual who wants to complete a project on the biodiversity of bird species in their local area.

Color Schema

The color scheme for the website includes:

  • Primary Color: #124a5e (Deep Teal) - This will be used for the header, navigation bar, and headings.
  • Accent Color: #fff (White) - This will be used for the background and content sections to ensure readability and a clean look.

Typography

The typography for the website includes:

  • Primary Font: Arial, sans-serif - This will be used for body text for readability.
  • Secondary Font: 'Helvetica Neue', Helvetica, Arial, sans-serif - This will be used for headings to provide a clean and modern look.
